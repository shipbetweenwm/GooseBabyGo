## 🤖 不只卖萌，还能帮你干活

**四种模式，想怎么用就怎么用：**

| 模式 | 一句话 | 适合场景 |
|:---:|:---|:---|
| 💬 **Ask** | 只聊天，不操作 | 学习、咨询、闲聊 |
| ⚡ **Craft** | 你说话，我直接做 | 写脚本、跑命令、改文件 |
| 📋 **Plan** | 先规划，确认后再执行 | 复杂任务、项目重构 |
| 👥 **Team** | 多个鹅组队协作 | 产品文档、方案讨论 |

### 📋 Plan 模式 · 复杂任务不翻车

鹅宝先拆解任务，你确认后才动手，每一步都看得见：

```
📋 重构数据库模块           [确认] [取消]
 ✅ 1. 分析现有数据库结构
 🔄 2. 设计新的表结构          执行中...
 ○  3. 编写数据迁移脚本
 ○  4. 运行测试验证
```

### 👥 Team 模式 · 一群鹅帮你打工

主管鹅拆任务 → 设计鹅/开发鹅/文案鹅并行执行 → 自动汇总报告

还支持**圆桌讨论**：多个角色围绕一个话题各抒己见，碰撞出最优方案。

> 🦆 一个鹅不够，那就来一群！
